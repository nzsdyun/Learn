    一些思考方法

    MECE mutually exclusive collectively exhaustive，彼此独立、完全穷尽
    逻辑思维、假设思考、框架思考

    - 咨询工作的一些特点

    成为咨询顾问的首要要素是：喜欢思考，因为这个工作大部分时候都需要思考，有好奇心探索未知的领域。另一个要素则是，思考体力。毕竟一整天即使是飞来飞去，也要不停的思考，可以说咨询是一个“我思故我在”的工种。而案例分析的面试环节更多考察也是这两点。

    咨询工作是否顺利，最关键的是能否取得客户的信任，获得更全面的信息，以及改革时的支持。而获取信任，需要有足够好的倾听能力，以及包容力。

    分析问题，指出问题所在仅是咨询工作的一部分，最重要的是建构能力，即“建立一个独一无二且实现时能带来足够震撼力的假设”的能力，以及“从零开始描绘并设计一个新提案”的能力。

    - 领导力

    领导能力是管理顾问最重要的技能，领导能力简单来讲，就是带领团队完成使命实现目标的能力。是以结果为导向的能力。领导力区别于项目管理的能力，不仅是推动事情顺利按流程展开，而更以提高结果质量为目的。

    全员都具有极强领导能力的组织，比只有一部分人具有领导能力的组织，更容易取得优异的成绩。
    拥有领导力的人，与“自己的意见”相比，会优先采用"能够达成目的“的意见。
    真正意义上的领导，是“为了达成团队的使命，制作有必要之事的人。
    将团队内各种各样的意见提取出来并加以总结，是领导最重要的工作。
    自己做出决定，承担这一决定可能带来的风险，并且向大家说明做出决定的理由。
    只有不断积累领导经验，才能够成为一个“可以取得优异成果的团队成员”
    只有在以“成果主义”为原则的环境下，才需要领导能力。

    这也是很多政府，事业单位的弊病，光有领导的岗位，无解决问题取得成果的领导能力。因为大家都无需对结果负责，只要不犯错就行。

    只有以提高成果为团队目标时，才需要领导能力。
    领导不是注重和谐的人，而是创造结果的人。
    只有选择在面临残酷的抉择和风险时能够做出正确判断的真正的领导，你猜能够活命。

    醍醐灌顶。结果永远是最重要的。这样才能真正做正确的事，先苦后甜，抓大放小。

    协调人在团队中非常关键，如果协调人同时身兼管理职务的话，团队的交涉将会进展得非常顺利。
    成员对领导的服从度，取决于“成员对取得成果这一目标的投入度以及重视程度”
    人只有不断接受挑战，才可能持续接触到新的事物，并且不害怕迎接更大的挑战。

    给予实践机会，赋权，明确目标，适当指导过程，让每个伙伴坚决的去承受挑战，才有可能快速成长。

    - 领导的四项核心任务

    1. 设定目标：较难实现但可实现的目标，并能让成员获得极大回报。只有这样才能鼓舞人心，并且获得成就感。比如阿里巴巴跨三个世界，成为第五大经济体等。

    2. 身先士卒

    3. 做决定。bad decision is better than no solution。当信息都充分的时候再做决定=不做决定。只是一种推卸责任的表现。领导的职责就在于没有充分信息和时间的情况下也能做正确的决定，只有这样，才有可能在竞争中获胜。也只有做出决定后，相关的问题才会浮现出来，从而解决这些问题，朝着目标更进一步。

    4. 传达信息。必须让团队成员对“成果”以及取得成果的路径有统一认知，这样才能在前进中获得最大的动能。每个人拥有不同的感受性和思考方式，所以当接触新情报和开始思考时，每个人都会产生出与他人不同的反应。这种情况反复出现的话，即便是向着同一个目标，但最终抵达的地点却有可能出现偏差。因此拥有领导力的人会不断重复，坚定的重复。

    设定目标，身先士卒，在前进道路上的关键点做出决定，保持与成员之间的沟通，这就是领导必须完成的四个任务。

    - 麦肯锡式领导能力学习

    1. 不断的强调要创造价值，即取得成果。在会议中一言不发就等于是浪费了回忆的时间，会被看做是工资蛀虫。

    2. 明确自己的结论。注重结论，而后才是过程。不应该浪费时间在与最终结论无关的细枝末节上，同时必须对影响结论的关键点进行仔细分析讨论。多问几句so what？让问题回归到关键点上，得出关键结论。只要养成专注于结论的习惯，就可以大幅缩短讨论所需要的时间，加快最终做出经营判断的速度。

    3. 做自己的领导。积极主动。只需要考虑如何（健康，而非宫斗，正直是基础）利用相关人员，使你的工作成果最大化。

    4. 在未完全具备能力之前就实战。在实战中体会，取得经验，总结得失，下次进步。

    5. 找到自己的领导风格。

    在日本的商业现场，有时候组织的和谐甚至比成果更加重要。特别是在大组织中，甚至有“绝对不能对其他部门的方针提出意见”的潜规则。很多人认为，就算失败也是那个部门的人自己负责，外人不能指手画脚。

    但实际上，就算身处不同的部门，提高企业整体的销售额和利润最大化这一成果目标，应该是全体员工共有的。大家都拥有这同一个目标，并且以实现这一目标为最优先的考量，那么当认为其他部门的判断存在问题的时候，就应该主动地提出来。

    但在绝大多数情况下，很多人都认为“这样做会引起摩擦”，不愿意对其他部门的决定（即便暗地里充满了不满和不安）表面上提出反对意见。当然，也有“不识相”的人总是对其他部门的问题提出自己的看法。

    对其他部门的问题保持沉默的人属于多一事不如少一事的类型，只有后者才是能够发挥领导能力的人。后者之所以对其他部门的问题提出自己的看法，既不是爱出风头管闲事，也不是不识相。唯一的原因就是他们知道“为了实现让这个企业的利益最大化的成果，就算没有人命令，也有责任和义务做自己应该做的事”。但日本的组织中，这样的人往往被看作是“破坏组织和谐的人”，结果遭到整个组织的疏远。

    P88-89

    只有选择在面临残酷的抉择和风险时能够做出正确判断的真正的领导，你才能够活命。…… 我们为了实现活命这个目标，可以牺牲其他的一切。

    可能这个人你平时就感觉难以相处，不愿和他接近，但你还是果断地选择乘上他的那条救生艇。因为就算性格不合、惹人讨厌，但只要他是一个“拥有判断力和决断力值得信赖，该说的话毫不犹豫地说，该做的事勇于承担风险去做”的人，那你就愿意搭乘他的船。

    当关系到重大成果的关键时刻所选择的领导，和没有目标时选择的“那家伙是个好人”“总是很努力，大家对他都很有好感的人”“在一起工作感到很快乐的人”“任何事情都处理得很完美很有能力的人”之类，是完全不同的概念。

    “领导能力与成果主义之间的关系”在日本并不被普遍理解。这也是很多人难以想象领导能力究竟是什么概念的根本原因。

    P99

    对于在时间和空间上有一段距离的人来说，只能看到领导创造的成果，却不会知道为了创造出这些成果，领导周围的人吃了多少苦。如果只看到成果，领导确实是值得称赞的对象。

    P100

    但领导将创造成果优先于组织的和谐。所以一个强有力的领导，对于同时代同空间的人来说，绝对不是“能够一起快乐工作的人”。

    成员对领导的服从度，取决于“成员对取得成果这一目标的投入度以及重视程度”。

    对领导持有负面态度的人，绝大多数都是“对取得成果不够投入的人”。对于不赞成领导的想法，认为领导想要达成的目标与自己没有关系的人来说，领导只是一个固执己见的独裁者罢了。

    …… 在日本做领导的人，之所以无法得到周围人的协助而孤军奋战，甚至还会遭到周围人的批判，就是因为其他人没有自己作为领导的体验，对领导的实际情况缺乏了解。所以这些人才会对领导的遣词造句、为人处世等表面上的行为提出这样那样的意见。

    救生艇的水手，没有让所有的乘客享受快乐漂流的责任。在他要做出决定时，也不必听取所有人的意见，或者经过大家讨论后决定。对他来说唯一重要的事情，就是尽可能多地拯救生命。

    或许有一部分乘客无法得到救援。但在这种时候不能有“既然不能全救，干脆大家一起死掉算了”的想法，坚持“或许会出现牺牲者，但要尽可能多地拯救生命”才是领导者应该做的。谴责“出现一个牺牲者就是领导失职”的人，既不是同乘一条救生艇的人，也不是自己身为领导的人。只要不能理解这一点，日本人就无法理解领导能力真正的意义。
